# 迭代进度跟踪

## 项目状态：✅ 全部完成，前后端已跑通

- 后端启动：`mvn spring-boot:run -pl file-manager-web`
- 前端启动：`cd file-manager-frontend && npm run dev`
- 一键启动脚本：`start.bat` / `stop.bat`
- API 文档：http://localhost:8080/api/doc.html
- 默认账号：`admin / admin123`
